\chapter{代码复现与数据来源说明}

本附录给出论文主要结果的复现方式、附件数据来源、输出文件对应关系以及核心代码清单。附录中的代码均通过\verb|\lstinputlisting|直接从\texttt{code/}目录引用，避免手工复制造成论文附录与实际运行代码不一致。

\section{运行环境与复现命令}

代码目录为\texttt{code/}，Python依赖由\texttt{uv}管理。进入\texttt{code/}目录后，完整复现顺序如下：

\begin{lstlisting}[style=appendixtxt]
uv run python scripts/run_all.py
uv run python scripts/run_award_audit.py
uv run python scripts/validate_outputs.py
uv run python -m compileall src scripts
uv run pytest
\end{lstlisting}

其中，\texttt{scripts/run\_all.py}生成五问基础求解结果，\texttt{scripts/run\_award\_audit.py}生成模型对照、敏感性分析和预测区间等增强证据，\texttt{scripts/validate\_outputs.py}检查关键表格、图件和约束条件是否满足。本文编译后正文为29页，参考文献与附录不计入正文页数。
